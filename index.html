Project Idea.

Overview

The Project will be mainly based on Cat based enjoyment, I don’t have anything concrete, but some sort of toy that can be put through stress to achieve a longer life span than normal cat toys. By using a Raspberry Pi to achieve a toy to help prevent loneliness in cats.

Motivation

The motivation for this project comes right from home, as I have a Ragdoll cat. They are very dog like creatures, so they do tend to show more affection than most cats. I have been told by friends that he’s very loud and annoying if there isn’t any one to hang around so I thought, how could I think up something unique that might cheer the little guy up when I’m gone.

Description

Either a ball type design with the raspberry Pi inside, connected to the raspberry Pi would be a motion sensor, a weighted motor set up to move different depending on the motion sensed by the sensor. Potentially I would like to add some sort of Bluetooth device or something to add more to a collar like device to enable more functionality between the cat and the toy, if possible. If a ball design is not possible I would like to move to something more stationary, a box like design where a ball goes in the top, goes down a path till it reaches the bottom, while allowing enough time for the ball to reach the end. Connected to the chamber where the ball is seated the ball is launched with a small burst of air made with a quiet hair dryer, or something of the sort. Also, when the ball is first placed in the contraption a small treat is dispensed on the device, as to promote learning for the pet and reward him, while he learns to operate the device by himself. The services that this would provide for the cat, while I would be away at work, as cat’s need at least 15 minutes of continuous attention.

Tools and Technologies

I would be using a Raspberry Pi to control the brain of the device. Connected to the raspberry pi would be a motion senser to detect whether the ball has been placed in the tube, part of an automatic feed device, as to not give out too many treats at once, at the bottom of the feed bowl is a pressure sensor that will fire the ball once pressure has been applied and released. I would be coding the raspberry pi to watch with the motion senser, releasing treats once activated then then waits for the sensor on the bowl to activate to use the motor device to shoot the ball for the cat to fetch and return to the box device.

Skills Required

I would need to learn how I how to connect everything to the Raspberry Pi and power it all with a small battery, while fitting the contraption in a box that is not to big to be an obstruction but provides enough space for air to escape. I would need to learn how to code for the Raspberry Pi. I will probably encounter problems figuring out how everything would work seamlessly in the way that I have envisioned

Outcome

The outcome of the project would be a trying to achieve a device that an animal can interact with and it will over time train the pet to use the toy for itself. The outcome would be cat owners would be able to buy this and entertain their cat, if they were out shopping, working a full time job, or just generally too busy to give their cat the attention it needs
